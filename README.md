# ChantXML
MusicXML for Gregorian Chant

## Resources

[LillyPond: musical score software that supports Gregorian chant](https://lilypond.org/examples.html#Early-Music)

[MusicXML is a widely used format](https://www.w3.org/2021/06/musicxml40/)

[The Gregorio Project: tools for typesetting Gregorian chant](https://gregorio-project.github.io/)

[Gregorio gabc: a notation for representing Gregorian chant using ASCII characters](https://gregorio-project.github.io/gabc/)

[GregorioTeX: a TeX style for typesetting scores](https://gregorio-project.github.io/gregoriotex/)

[GregoWiki: an information site for Gregorio-related resources](http://gregoriochant.org/) (See many more resources listed here)

[Illuminare Score Editor: An online editor for Gregorian scores using gabc](http://dev.illuminarepublications.com/gregorio/)

[Gregorio chant engraver](https://run.gregoriochant.org/cgi/process.pl)

[The Source and Summit chant editor, which provides results similar to Gregorio](https://www.sourceandsummit.com/editor/alpha/)

[GregoBase: A database of gabc scores to use with Gregorio](http://gregobase.selapa.net/)
